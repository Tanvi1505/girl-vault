# girl-vault
pinterest for your "i'll buy this later" links. 
okay so here's the thing
you're scrolling at 1am. you find THE dress. perfect. you're not buying it right now because you're broke / it's late / you need to think about it. so what do you do?

add to cart? you either impulse buy it or forget it exists
pinterest? it saves the vibe, not the actual link to buy the thing
notes app? lol you're never finding that again
browser bookmarks? that folder has 400 things and zero organization

there was genuinely no good solution for this. so i built one. it's called the girl vault and it does exactly one thing — saves the link so you can come back to it when you're ready.

---------------------------------

what it actually does:
paste a link. it's saved. that's literally it.
no sign up. no account. no form to fill out. just paste.
then when the link loads in the background, it pulls the preview image and title automatically so your vault looks like an actual pinterest board, not a sad list of urls.
collections — you decide how to organize. want a "dresses" collection? create it. "february wishlist"? also valid. "things i will never afford but still"? same. you name it, it exists. nothing is preset because your brain works differently than mine.
everything unsorted just sits in unsorted (like raindrop.io does it) until you decide to move it somewhere — or you never do, totally fine.

-----------------------------------

how to use it for personal use
option 1 — download and just open it locally:

download the index.html file from this repo
open it in your browser (just double click the file)
start pasting links
your saves stay in your browser storage — they're there every time you open the file

that's it. no setup. no wifi needed after that first open. it just works.
option 2 — host it on github pages (free):

fork this repo
go to your repo settings → pages → source: main branch
it goes live at yourusername.github.io/girl-vault
bookmark that url on your phone and use it like an app

option 3 — literally just steal the file:
download index.html, put it on your desktop, open it. zero commitment, maximum function.
